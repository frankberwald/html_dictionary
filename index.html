Dicionário HTML

    Estrutura

  • header = Cabeçalho
  • p = Parágrafo
  • footer = Rodapé
  • nav = Barra de navegação
  • main = Área do conteudo principal
  • article = Area de publicação
  • section = Area agrupada
  • aside = Barra lateral
  • ul = Lista simples
  • span = Destaca uma parte do texto
  • style = css direto na pagina, sem utilizar css/style.css
  • br = pula linha

    Form e Inputs

  • input:tipo = atalho para definir a input com tipo sem a necessidade de escrever o comando por extenso
  • text = texto simples
  • email = endereço de email
  • password = senha com caracteres ocultos
  • url = só aceita links
  • datalist = lista de opções
  • number = numeros simples
  • textarea = textos mais extensos, como uma caixa de mensagem
  • rows="numero" cols="numero", define o limite de colunas e linhas
  • select = lista dropdown de itens
  • option = deve ser colocado apos select < select name="produto" > < option value="opção1> < /select >
  • multiple size="numero" = define um limite para a lista
  • date = selecionador de datas
  • checkbox = caixas de marcação com opções múltiplas
  • radio = caixas de marcação com opçôes individuais
  • checked = imprime na tela a caixa marcada
  • file = upload de arquivos
  • submit = botão de enviar
  • range = slider pra selecionar intervalo
  • Tick = range com valores fixos
  • color = selecionador de cores
  • placeholder = dentro do comando "< input type "text" placeholder"Insira seu nome" > = pre dispõe um texto dentro da input
  • required = dentro do comando "< input type "text" placeholder"Insira seu nome" > = mostra uma mensagem de aviso como = Preencha o campo para continuar e evita de enviar campos vazios
  • minlength/maxlength = define um limite para os caracteres inseridos na input "< input type "tipo" minlength="3" >
  • disabled = desativa qualquer campo input = "< input type "text" disabled>
  • value = pre define um valor
  • action = define uma ação proveniente dum script
